# 🎬 Movie Dataset Analysis using SQL

## 📌 Project Overview
This project focuses on analyzing a movie dataset using SQL to extract meaningful insights. The analysis includes data exploration, data cleaning, and solving real-world business questions using structured queries.

The goal of this project is to demonstrate practical SQL skills required for a Data Analyst role.

---

## 🎯 Business Problems Solved
- What is the average rating of movies vs TV shows?
- How has content production changed over time?
- Which age certifications receive higher ratings?
- What is the average runtime of different content types?
- Which shows have the highest number of seasons?
- How many titles are released each year?

---

## 🧠 Key Insights
- Content production has significantly increased in recent years  
- Movies and TV shows differ in both runtime and audience ratings  
- Age certification has a noticeable impact on content ratings  
- A small number of shows dominate in terms of total seasons  
- Recent decades have seen a sharp rise in content creation  

---

## 🛠️ Tools & Technologies
- SQL (MySQL)

---

## 📊 Analysis Performed
- Data exploration using SELECT, WHERE, and LIMIT  
- Data cleaning (handling NULL and empty values)  
- Aggregations using AVG(), COUNT(), SUM()  
- Grouping and filtering using GROUP BY and HAVING  
- Trend analysis using time-based grouping
