# 🎬 Movie Dataset Analysis using SQL

## 📌 Project Overview
This project analyzes a movie dataset using SQL to extract meaningful insights. It includes data exploration, data cleaning, and analytical queries.

## 📊 Key Analysis
- Average ratings of movies vs shows
- Content production trends over decades
- Age certification impact on ratings
- Runtime comparison
- Top shows by seasons
- Titles released per year

## 🛠 Tools Used
- SQL (MySQL)

## 📈 Key Insights
- Content production has increased in recent years
- Movies and shows differ in runtime and ratings
- Age certification affects audience scores

## 📂 Files
- SQL queries and explanations (Word document)
